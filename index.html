DMV -- Reactjs/Microsoft XRM/Material UI/Redux/ Full CRUD

This DMV app was created with a team of three other people, check them out in the github contributors page! We created a Microsoft XRM solution from scratch and used that as the backend for our api calls. Material ui makes up the design, every example of crud is displayed for each entity, and almost every state change in the entire site, from form views to field inputs and rendering is handled in Redux. Check it out on Github here.

Book Face -- Reactjs/SQL/Authentication/Bcrypt/ Full CRUD

Book Face is a social media site I made that does everything a social media site should be able to do. Users can make an account, login to that account anytime after, make posts, edit their own posts, and comment on theirs and other peoples posts. I even implemented some very rough chat functionality that works with socket io so users can send messages back and forth! Check it out on GitHub here.

Sexy Board Games -- Reactjs/SQL/Authentication/Bcrypt/SocketIo

Sexy Boardgames is another group app I built in a team environment. Users are able to login, create an account, and play chess and tictactoe against each other. The connection between the two users is once again made with socket io. Check it out on GitHub here.
